# CodeExpedetion Portfolio 🚀

A modern portfolio website built using **React + Vite**, styled with **TailwindCSS v4**, and powered by **React Router** with animations and React-Icons.

---

## ✨ Features

- ⚡ Fast performance with **Vite**
- 📱 Fully responsive layout (mobile / tablet / desktop)
- 🎨 Modern UI with **TailwindCSS v4**
- 🧭 Page routing with **React Router**
- 🎞️ Smooth animations (React Animate / animation utilities)
- 🎯 Icons powered by **React Icons**
- 🧩 Component-based clean structure

---

## 🛠️ Tech Stack

- **React**
- **Vite**
- **TailwindCSS v4**
- **React Router DOM**
- **React Icons**
- **React Animate**

---

